# Aplicación Móvil (App)
La aplicación móvil está construida con **React Native** y **Expo**. Sirve como interfaz principal para los familiares o cuidadores.

## Funcionalidades Principales
- **Mapa en Tiempo Real**: Visualización del bastón en un mapa interactivo.
- **Notificaciones**: Alertas push si el usuario sale de una zona segura o si la batería está baja.
- **Historial de Rutas**: Consulta de recorridos anteriores.

## Tecnologías
- React Native
- Expo
- Firebase (Autenticación y Base de Datos)
