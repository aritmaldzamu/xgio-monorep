# Bienvenidos a XGIO 🦯

**XGIO** es un Sistema de Rastreo GPS en Tiempo Real para un Bastón Inteligente, diseñado para mejorar la autonomía y la seguridad de las personas con discapacidad visual.

---

## 🎯 Objetivo del Proyecto

Brindar una herramienta tecnológica accesible que permita a los usuarios con discapacidad visual desplazarse con mayor confianza, mientras sus cuidadores pueden monitorear su ubicación en tiempo real mediante una aplicación móvil.

## 🏗️ Arquitectura del Sistema

El ecosistema de XGIO se compone de tres pilares fundamentales que interactúan en tiempo real:

1. **[📱 Aplicación Móvil (App)](app.md)**: Interfaz para los cuidadores, desarrollada en React Native, que muestra mapas, historiales de ruta y alertas.
2. **[⚙️ Backend (Servidor)](backend.md)**: Lógica de procesamiento de datos y analíticas de rutas, alojada en Vercel y Firebase.
3. **[🔌 Firmware (Hardware)](firmware.md)**: El "cerebro" del bastón inteligente, programado en ESP32, encargado de leer el GPS y comunicarse con la nube.
