# Hardware (Firmware)
El firmware es el código incrustado en el microcontrolador del bastón inteligente.

## Funcionalidades Principales
- **Lectura GPS**: Obtiene latitud y longitud del módulo GPS constantemente.
- **Sincronización Cloud**: Envía los datos a Firebase en tiempo real.
- **Gestión de Energía**: Entra en modos de bajo consumo cuando el bastón no está en uso.

## Tecnologías
- C++ (Arduino Core)
- ESP32
